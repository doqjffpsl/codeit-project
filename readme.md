# 📢 AI 광고 콘텐츠 생성 서비스

## 1. 프로젝트 소개

본 프로젝트는 생성형 AI 기술을 활용하여 소상공인이 손쉽게 광고 콘텐츠를 제작할 수 있도록 지원하는 서비스입니다.

사용자는 상품 이미지를 업로드하고 원하는 광고 스타일을 선택하면, AI가 해당 이미지를 분석하여 상품 특징을 설명하고 광고 문구를 자동으로 생성합니다.

디자인이나 마케팅 경험이 없는 소상공인도 간단한 입력만으로 광고 문구를 생성할 수 있도록 설계되었습니다.

---

## 2. 주요 기능

###  1) 상품 이미지 분석
- 업로드된 이미지를 기반으로 상품 특징 자동 분석

###  2) 광고 문구 생성
- 선택한 스타일(감성 / 고급 / 귀여움 / 할인 강조 / SNS용)에 맞는 광고 문구 3개 생성

###  3) 이미지 업로드 기반 서비스
- 사용자가 직접 상품 이미지를 업로드하여 결과 생성

---

## 3. 기술 스택

- Python
- Streamlit
- OpenAI API (GPT-5-mini)
- Pillow (이미지 처리)
- python-dotenv (환경 변수 관리)

## 5. 실행 방법

### 1) 저장소 클론
```bash
git clone <repository-url>
cd project-folder
```

### 2) 패키지설치
pip install -r requirements.txt

### 3) .env파일 설정
프로젝트 루트에 .env 파일 생성 후 아래 내용 입력:
OPENAI_API_KEY=본인의_API_키

### 4) 실행
streamlit run app.py
